## Scoring
Start at 10, deduct:
- **-5**: XSS, hardcoded secrets, eval(), SQL injection
- **-2**: Insecure deps, missing auth, open redirects
- **-0.5**: Security hardening opportunities
- **+1**: Good patterns (sanitization, secure auth, deps)

Minimum: 1
